## PGP 公钥验证

译者个人 PGP 公钥（有效期至 2029/7/12），用来检验 pdf 文件是否由本人签名释出：

- **公钥指纹**：75E0 C321 BDDB 6EDF 99AC  6219 5B92 1034 CA2E D075

### PGP 验证方法

1. 下载 Github Release 中的 `book_signed.pdf.gpg` 文件
2. `gpg --verify book_signed.pdf.gpg`（需要安装 gpg）
3. 核对 EDDSA key 一栏的大写字母字符串是否匹配以上公钥指纹

## PDF 官方版本验证（X.509 自签名证书）

- **颁发者：** natsunoyoru97@outlook.com
- **签名算法：** SHA256RSA
- **证书指纹：**
  - 无分隔符：`b1a81117c017f8fee48cdb7adbcc304ff140cc08`
  - 带分隔符：`b1:a8:11:17:c0:17:f8:fe:e4:8c:db:7a:db:cc:30:4f:f1:40:cc:08`
- **有效期**： 2026/7/13-2027/7/13

### 验证方法

1. 用 PDF 阅览器（如 Adobe Acrobat Reader、WPS 和 foxit）打开释出的 pdf 文件
2. 点击签名面板查看签名详情
3. 核对**颁发者**和**证书指纹**是否与上表一致
4. 在签名详情中查看**签名算法**，应为 `SHA256RSA`

### 验证结果解读

| 验证结果 | 含义 |
| :--- | :--- |
| ✅ 指纹匹配 + 签名算法为 SHA256RSA + 签名状态为"有效"或"未知" | **文件可信**，未经篡改 |
| ✅ 指纹匹配 + 显示"签名有效性未知" | **正常现象**（自签名证书），防篡改功能同样生效 |
| ❌ 指纹不匹配 或 显示"签名无效" | **文件已被篡改或来源不可信**，请勿使用！ |
